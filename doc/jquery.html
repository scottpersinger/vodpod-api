JQuery plugin

Here's a javascript object with some helper functions for working with the API. It uses jQuery for AJAX handling, but any library will do.

Download the toolkit

In general, you'll be calling get() with some path, options, and callbacks to be called on response.

vodpod.api_key = 'a3782f...'; vodpod.get('/me', {include: 'videos'}, {success: function(user) { alert(user.name + "'s first video is " + user.videos[0].title); }});

You can pass the path as an array of strings, which will be automatically joined. You can also specify an error callback, which will be invoked when an API error or connection problem occurs. Finally, you can set default options to include with every request by changing vodpod.options.
