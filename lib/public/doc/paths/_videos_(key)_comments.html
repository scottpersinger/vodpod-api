API: /videos/<key>/comments

A list of the Comments associated with a video. All collected versions of a video have the same comment set for this method. If you are looking for the comments made only on this collection's version of the video, see /users/<user>/collections/<collection>/videos/<key>/collection_comments.

Parameters

api_key
API key.
include
Which attributes will be included in the result set. Some attributes are always provided.
limit
The number of results to return at a time. Limit and per_page are synonyms. If both are provided, limit takes precedence.
offset
How far into the result set to begin returning objects. Offset 0 returns records 0, 1, 2, ... offset 2 returns records 2, 3, 4, ...
order
One of asc or desc, for ascending and descending sort orders, respectively.
page
The page of the results to return. If offset is provided, it takes precedence over page.
per_page
The number of results to return at a time. Limit and per_page are synonyms. If both are provided, limit takes precedence.
sort
The name of an attribute to sort by. Only certain columns are sortable; see the object's documentation for details.
