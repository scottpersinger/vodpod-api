API: /users/<user>/following

A list of users this user is following.

Parameters

api_key
API key.
include
Which attributes will be included in the result set. Some attributes are always provided.
limit
The number of results to return at a time. Limit and per_page are synonyms. If both are provided, limit takes precedence.
offset
How far into the result set to begin returning objects. Offset 0 returns records 0, 1, 2, ... offset 2 returns records 2, 3, 4, ...
page
The page of the results to return. If offset is provided, it takes precedence over page.
per_page
The number of results to return at a time. Limit and per_page are synonyms. If both are provided, limit takes precedence.

Example

/users/aphyr/following?limit=2
