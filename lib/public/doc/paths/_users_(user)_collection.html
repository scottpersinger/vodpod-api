API: /users/<user>/collection

Alias for /user/<user>/collections/<collection>, where <collection> is the default Collection for <user>.

Parameters

api_key
API key.
include
Which attributes will be included in the result set. Some attributes are always provided.

Example

/users/aphyr/collection
XMLJSON
<?xml version="1.0" encoding="UTF-8"?>
<collection>
  <description/>
  <key>aphyr</key>
  <name>aphyr's videos</name>
  <videos_count>348</videos_count>
</collection>

[
  true,
  {
    "name": "aphyr's videos",
    "videos_count": 348,
    "description": null,
    "key": "aphyr"
  }
]
