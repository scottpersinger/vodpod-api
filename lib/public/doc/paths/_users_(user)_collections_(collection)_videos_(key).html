API: /users/<user>/collections/<collection>/videos/<key>

Returns a specific Video in the context of a collection. Instead of the canonical title and description, returns the title and description the collector assigned. key is the numeric video key.

Parameters

api_key
API key.
include
Which attributes will be included in the result set. Some attributes are always provided.

Example

/users/spencer/collections/electro/videos/381539?include=tags
