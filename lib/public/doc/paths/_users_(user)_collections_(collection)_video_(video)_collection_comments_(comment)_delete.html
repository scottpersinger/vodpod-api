API: /users/<user>/collections/<collection>/video/<video>/collection_comments/<comment>/delete

Delete a specific CollectionVideoComment by key. An authorization key is required. This method must be called with an HTTP POST, and returns true when successful.

Parameters

api_key
API key.
auth_key
Auth key.

Example

/users/aphyr/collections/aphyr/video/143423/collection_comments/28542/delete?auth_key=xxx
