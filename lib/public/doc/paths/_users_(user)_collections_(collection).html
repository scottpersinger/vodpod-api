API: /users/<user>/collections/<collection>

Describes a specific Collection belonging to a User.

Parameters

api_key
API key.
include
Which attributes will be included in the result set. Some attributes are always provided.

Example

/users/spencer/collections/spencerpod
XMLJSON
<?xml version="1.0" encoding="UTF-8"?>
<collection>
  <description>Spencer's favorite viral videos</description>
  <key>spencerpod</key>
  <name>SpencerPod</name>
  <videos_count>528</videos_count>
</collection>

[
  true,
  {
    "name": "SpencerPod",
