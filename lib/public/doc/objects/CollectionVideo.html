API: CollectionVideo

A CollectionVideo is a specific version of a Video. When a user collects a video, either from some external video host (youtube) or from some video already on Vodpod, a CollectionVideo is added to one of their Collections.

Just like Videos, CollectionVideos include an embed code to view the video in an HTML document, a thumbnail (provided by the Vodpod thumbnailer service or the original host), and assorted metadata. CollectionVideos have multiple Comments, belong to one Collection and User, and may have several Tags.

Attributes

autoplay_embed

Like embed, but with flash variables set so as to play the video immediately upon loading.

collected_from_url

The URL that this video was collected from.

collection

The Collection this video is a part of.

collection_comments

An array of Comments specifically on this CollectionVideo. Unlike comments, collection_comments is not shared with any other versions of a video. It is typically only set for Vodspot use.

collection_comments_count

The total number of collection comments on this collection video.

collections_count

The number of collections this video belongs to.

comments

An array of Comments on this video. Comments are shared between all CollectionVideos and Videos with the same key.

comments_count

The number of comments on this video.

created_at

the date and time this video was created at.

description

The original description from the video provider.

embed

The HTML embed tag for displaying this video.

embed_src

The src attribute for the HTML embed tag.

key

The numeric ID of the video.

media

The URI of the raw media file for this video.

media_240_mobile

The URI of a 240p mobile-ready MP4.

media_480_web

The URI of a 480p web-ready MP4.

ranking

The user-defined ordering for videos. Sorted in descending order by default. You can pass sort=ranking to get the same order used on vodpod.com.

recommends

The number of users who have recommended this video. Recommends are distint from collects.

tags

A collection of Tags on this video.

tags_count

The total number of tags associated with this video.

thumbnail

A URL to the default thumbnail (100px wide).

thumbnail_160

A URL to a medium thumbnail of the video (160px wide)

thumbnail_320

A URL to a large thumbnail of the video (320px wide)

thumbnail_80

A URL to a small thumbnail of the video (80px wide)

title

The human-readable title of the video.

total_external_views

The total number of views for this video from sources outside Vodpod.

total_views

The total number of views for this video.

url

A WWW URL to watch the video.

user

The user who collected this video.

video_host

Information about the original hosting site. Includes three attributes: - description: The host's description of the video. - domain: The domain we obtained the video from. - url: The URL to view this video at the original host.

weekly_views

The number of views this week for this video.
